Используйте поле ввода
для коротких текстовых

Если вводимое значение имеет определенный формат, например, дд.мм.гг., используйте специальную версию поля
для ввода даты.

Название

Название поля пишется с заглавной буквы.

Называйте поле ввода существительным, указывающим что нужно ввести.

Неправильно

Не ставьте двоеточие после названия: без него все понятно, оно только создает лишний визуальный шум.

Неправильно

Большинство полей ввода используют название. Название размещается сверху (для форм, где требуется последовательное заполнение полей, например, регистрация) или слева (для форм, где заполняются всего несколько полей, например, настройки).

Размер и расположение

Ширина поля должна соответствовать ожидаемому количеству информации. Это подсказывает пользователю в какой форме от него ждут информацию, помогает пользователю убедиться, что он правильно заполнил поле.

В рамках одной формы используйте 1-4 ширины для полей.

Поля ввода можно использовать и без зеголовков.

Если название расположено слева, оно должно выравниваться по базовой линии текста в поле.

Название поля в несколько строк должно выравниваться по базовой линии
первой строки.
