Когда использовать

Чекбокс используется для управления параметром с двумя состояниями. Используйте его для выбора элементов списка. Чекбокс не запускает действие немедленно. Как правило для этого нужно нажать подтверждающую кнопку. Для немедленного включения режима в интерфейсе лучше подходит тогл.

Валидация

Валидация может быть только в том случае, если обязательно выбрать хотя бы один вариант. Чекбоксы с валидацией выделяются красной рамкой.
