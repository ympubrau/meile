Когда использовать

Раскрывающийся список позволяет выбрать значение из заранее известного набора вариантов.При нажатии на кнопку раскрывающегося списка появляется список вариантов для выбора.

Ширина кнопки должна быть такой, чтобы большая часть значений из списка помещалась полностью, если выбранное значение не помещается, появляется многоточие.

Выберите длинный очень текст, он даже в селект не влазит!
